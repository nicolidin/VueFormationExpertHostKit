Petit repo d’hôte pour une formation Vue niveau expert : la lib Vue vit en submodule (`submodule/vue-lib-submodule-kit`), avec de l’atomic design dedans. Rien de plus à raconter ici.

Pour cloner avec la lib : `git clone --recurse-submodules <url>` (sinon `git submodule update --init --recursive`).
